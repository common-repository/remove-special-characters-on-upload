== Description ==
A plugin that will help you remove special characters from the name of your files.

## Descrição ##
Um plugin que irá remover os caracteres especiais de nomes de arquivos no upload.

== Installation ==
Just install and enjoy plugin. This plugin works with the native upload function.

## Instalação ##
Apenas instale o plugin e aproveite! Ele irá atuar silenciosamente no upload nativo de arquivos
